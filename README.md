# Como executar o projeto Quiz

Este projeto foi desenvolvido utilizando **Flutter** e pode ser executado através do **Visual Studio Code (VS Code)**.

## 1. Instalar o Visual Studio

Antes de instalar o Flutter, é necessário ter o **Visual Studio Community** instalado no computador.

1. Acesse o site oficial da Microsoft.
2. Baixe o **Visual Studio Community** (versão gratuita).
3. Instale normalmente no seu computador.

## 2. Instalar o Flutter no VS Code

1. Abra o **Visual Studio Code**.
2. Pressione **Ctrl + Shift + P** para abrir a paleta de comandos.
3. Digite **Flutter**.
4. Clique em **Flutter: New Project**.
5. O VS Code solicitará o local do **Flutter SDK**.

### Baixando o SDK

1. Clique em **Download SDK**.
2. Escolha a pasta onde deseja instalar o Flutter.
3. Clique em **Clonar Flutter**.
4. Aguarde o download terminar (pode levar alguns minutos).

Após a instalação ser concluída, **reinicie o computador** para finalizar a configuração.

## 3. Baixar o projeto

1. Baixe o arquivo **quiz.zip**.
2. Extraia o conteúdo do arquivo.
3. Entre na pasta extraída.
4. Copie ou mova a pasta **quiz** para um local de fácil acesso no computador.

⚠️ Importante:
Não execute o projeto diretamente dentro do **arquivo .zip**, pois isso pode causar erros.

## 4. Abrir o projeto no VS Code

1. Abra o **Visual Studio Code**.
2. Clique em **File → Open Folder**.
3. Localize a pasta **quiz**.
4. Selecione a pasta e clique em **Selecionar Pasta**.

O VS Code irá carregar todos os arquivos do projeto.

## 5. Executar o projeto

1. No VS Code, clique em **Terminal → New Terminal**.
2. No terminal que abrir, digite:

```
flutter run
```

3. O sistema solicitará que você escolha um **browser** para executar o projeto.
4. Selecione o navegador desejado.
5. Aguarde alguns segundos até que o **Quiz seja aberto automaticamente**.

## Pronto!

O projeto será executado e você poderá utilizar o sistema normalmente.
