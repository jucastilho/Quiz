Você deve instalar o flutter para conseguir rodar o código no Visual Studio Code (logo azul).

Para instalar o flutter, você precisa ter o Visual Studio (logo roxo) instalado no seu computador. 
Caso não tenha, instale o Visual Studio Community gratuitamente pelo site da Microsoft.

Após isso, entre no VS Code (azul) e pressione Control + Shift + P. Na paleta de comandos, digite flutter. Selecione Flutter: Novo Projeto.
O VS Code solicitará que você localize o SDK do Flutter no seu computador. Selecione " Baixar SDK".
Quando a caixa de diálogo Selecionar pasta para o SDK do Flutter for exibida, escolha onde deseja instalar o Flutter.
Clique em Clonar Flutter.
O download será iniciado e pode levar alguns minutos.
Após a instalção total, reinicie o computador para que a configuração seja feita por completo.
Agora o flutter já está instalado.

Baixe a pasta presenete aqui - quiz.zip. 
Quando já baixada, entre nela, tire a pasta "quiz" de dentro da pasta baixada (pastas zipadas não funcionam).
Coloque a pasta 'quiz' no Acesso Rápido, para facilitar.

No VS Code, clique em File -> Open Folder.
Localize a pasta quiz e clique nela até aparecer as pastas do código.
Quando as pastas do código aparecerem, clique em "Selecionar Pasta".
Abrindo a pasta, clique em Terminal -> New Terminal.
No terminal digite flutter run. Logo após isso, o sistema pedirá para selecionar o Browser de sua escolha. Selecione e aguarde o Quiz abrir.
